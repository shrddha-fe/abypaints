# ColorKids

A kids coloring app for iPhone and iPad — tap-to-fill, brush paint, themed outline pages, and an in-app gallery.

## Open & run

1. Install **Xcode** from the Mac App Store (full Xcode, not only Command Line Tools).
2. Open `ColorKids/ColorKids.xcodeproj`.
3. Select an iPhone or iPad simulator.
4. Set your **Team** under Signing & Capabilities (needed for device/App Store).
5. Press **Run** (▶).

## What’s included

| Feature | Detail |
|--------|--------|
| Fill tool | Tap a region to flood-fill |
| Brush tool | Paint freehand with size slider |
| Themes | Space, Unicorns, Wizard Academy, Kawaii Friends, Cute Kitty Club |
| Pages | 20 built-in outline drawings |
| Save | In-app **My Gallery** (device storage) |
| Devices | iPhone + iPad |

## App Store note (important)

Themes are **original** and only *inspired by* popular vibes. Do **not** use official Harry Potter, Sanrio, or Hello Kitty names, logos, or characters without a license — Apple will reject trademarked content.
